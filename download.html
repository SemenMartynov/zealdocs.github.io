Download

There are various ways to download Zeal, depending on which operating system you use, and whether you want to use pre-compiled binary, or build it yourself from source code.

Windows

Stable release

A 32-bit version for Windows is available in form of installable package (MSI) or as a portable edition.

Download installer (36.8 MB) Download portable archive (Coming soon)

CI builds

For a real cutting edge experience you can try our continuous integration builds, which are automatically created on every change in our Git repository. Keep in mind that CI builds are not intended for every day use and they will crash even more than stable releases. You have been warned!

View CI builds

Linux

Prebuilt packages are provided for variety of Linux distributions. Some of the packages are maintained by Zeal developers, while others are courtesy of independent contributors.

Arch Linux

Arch Linux users can easily obtain Zeal from AUR. There are two packages available: zeal provides the latest stable release, while zeal-git builds unstable source code from our Git repository.

View zeal in AUR View zeal-git in AUR

openSUSE

Zeal is part of the official openSUSE software repositories.

View openSUSE package

Ubuntu

Ubuntu users can enjoy our own PPA repository. Run the following commands to add the PPA to your system:

$ sudo add-apt-repository ppa:zeal-developers/ppa
$ sudo apt-get update
$ sudo apt-get install zeal
Attention non-Unity users! Due to a bug in appmenu-qt5 package Zeal main menu might be missing on desktop environments other than Unity. A simple solution is to remove appmenu-qt5 package from your system.

View PPA page

Source code

Complete source code is available on GitHub. To build Zeal follow instruction in the README.md file.
